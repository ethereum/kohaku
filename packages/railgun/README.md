# @kohaku-eth/railgun

Cross-platform Railgun Typescript SDK with support for kohaku's standard interfaces.  The SDK is 
designed to interface with railgun's on-chain contracts, facilitating state syncing, transaction 
construction, proof generation, transaction submission via broadcasters, and proof submission.

## Installation

```sh
pnpm add @kohaku-eth/railgun
```
