# railgun-rs

Railgun-rs is a Rust Railgun client library, designed to interface with railgun's on-chain and 
off-chain components. It provides support for:
 - UTXO and TXID syncing
 - State querying
 - Transaction construction
 - Proof generation
 - POI proof generation
 - POI proof submission
 - Transaction submission via broadcasters

## Examples

For examples of how to use railgun-rs, see the [integration tests](./tests/integration/).

## Overview

This library provides high-level interfaces for interacting with the Railgun protocol.
