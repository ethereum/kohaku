# Contributing

First off, thank you for considering contributing!

## Engineering Practices

This project adopts the [Google Engineering Practices Documentation](https://github.com/google/eng-practices) as our standard for code reviews and general engineering practices. Please familiarize yourself with these guidelines to understand our expectations during code review.

## Project Guidelines

*   **Package Manager:** We use `pnpm` for managing JavaScript and TypeScript dependencies. Please ensure you are using `pnpm` to install dependencies and run scripts.
*   **Directory Rules:** Be sure to follow the instructions in any `AGENTS.md` files located in the directories you are working on.
